Привет мир!

Этот проект создан, чтобы протестировать работу GitHub

Основные команды git:

  1. git --version - Проверить версию
  2. git config --global user.name "Olga" - Задать имя пользователя
  3. git config --global user.email "olga@mail.ru" - Email пользователя
  4. git config --global core.autocrlf true - Единый формат конца файла
  5. git init - Инициализация git
  6. git status - Проверка статуса
  7. git add - Добавление измененных файлов в Staged area
